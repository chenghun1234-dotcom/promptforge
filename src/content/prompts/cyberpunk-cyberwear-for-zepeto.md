## Introduction to Cyberpunk Cyberwear for Zepeto
Cyberpunk cyberwear is a popular trend in the world of Zepeto, a social media platform where users can create and customize their own avatars. As a professional creator, you can capitalize on this trend by designing and selling unique, futuristic outfits that appeal to Zepeto users.

## Crafting the Perfect Prompt
To create stunning cyberpunk cyberwear using AI, you'll need to craft a well-structured prompt that guides the model towards your desired outcome. The main prompt should include key elements such as:
* Futuristic inspiration
* Neon-lit accents
* Metallic textures
* Sleek, high-tech accessories

The negative prompt should exclude unwanted features, such as:
* Overly simplistic designs
* Bright primary colors
* Lack of futuristic elements

## Technical Settings for AI Generation
To achieve the best results, use the following technical settings:
* **Sampler:** Euler a
* **CFG:** 7.5
* **Steps:** 50

These settings will allow the AI model to generate high-quality, detailed images that meet your requirements.
